The Tech Academy Live Project

This project known as "The Tech Academy Live Project" is one of the final courses in the The Tech Academy's bootcamp. HERE you can find my additions to the project.
Summary:
The Live Project is an MVC Code-First web application. I, as well as many other attendees of the Tech Academy, used this project as a way to get experience working as a group on a singular project. We used AzureDevOps to keep the group organized, and instructors led us through what needed to be done. The project itself is an application the bootcamp uses to keep track of the hiring status of soon to be graduate attendees by tracking application statuses and overall readiness.
